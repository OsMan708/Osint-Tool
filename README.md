# Osint-Tool
🇬🇧 English

Osint Tool is a versatile console utility for Open Source Intelligence gathering. It features a built-in localization system and pre-integrated modules.
✨ Features

    Sherlock (Integrated): Find social media accounts by username.

    IP Lookup: Detailed geolocation and ISP analysis.

    Phone Lookup: Identify carrier and region data.

    WHOIS Domain: Retrieve registration details.

    My IP / VPN: Security check for your current connection.

    Nickname Gen: Generate secure aliases.

🚀 How to Run

    Ensure Python 3.11.9 is installed.

    Run the tool:

    python osint_tool.py
